# VoyageBlitz

A warm, emotionally-aware travel planning app that helps you understand what kind of trip you actually want.

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Set up Firebase credentials in `.env.local`

3. Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

## MVP Features

- Reflective travel survey
- Google authentication
- AI-powered trip interpretation
- Personalized destination comparison board
- Theme selection based on travel psychology
